To Auto-assign Reviewer(s) to specific Categories:
1. Select a Category from the dropdown menu.
2. Select the Reviewer from either the Auto-assigned Reviewers or Available Reviewers.
3. Click Add button or Remove button from that specific Category.
4. Click Save button.

To add or remove a Sidetab to or from a Category:
1. Select a Category from the dropdown menu.
2. Select the Sidetab from either the Assgined Side-tabs table or Available Side-tabs table.
3. Click Add button or Remove button from that specific Category.
4. Click Save button.

Overview

The primary goal of the MPA Engineering Response Desk Portal is to:

  • provide a system that allows users to file requests and reviewers to monitor, address, and respond to those requests.
  • provide support for project tracking through various procurement and construction phases, including separate functions for both project administrator and project manager.

The following functions are available:

  • Request Report Tab
    • View, filter and print report for total number of requests by category.
    • View, filter and print report for total number of requests by requester.
    • View, filter and print report for total number of assigned/pending requests by reviewer.
    • View, filter and print report for minimum/average/maximum response time by category.
    • View, filter and print report for minimum/average/maximum response time by reviewer.
  • Admin Tab
    • View, add and remove a user to or from a role.
    • Create and edit a request category.
    • View, add and remove a sidetab to or from a request category.
  • Assignment Rules Tab
    • Auto-assign reviewer(s) to specific categories.
  • Assignments Tab
    • View a list of submitted requests by default.
    • Filter and search requests based on various criteria (i.e. description, requester, type, status, date, etc.)
    • View request details page by clicking on the sequence number.
      • Mark a submitted request as high priority in request general information.
      • Assign a submitted request to one or more reviewers.
      • Receive automated emails notifying the assigner that a new request has been submitted or auto-assigned.
      • Communicate to requesters and assigners via tracked comments by automated emails.
  • Reviews Tab
    • View a list of assigned requests by default.
    • Filter and search requests based on various criteria (i.e. description, requester, type, status, date, etc.)
    • View request details page by clicking on the sequence number.
      • View request general information.
      • Assign a submitted request to one or more reviewers.
      • Receive automated emails notifying the reviewer that they have been assigned to a request.
      • Receive automated emails notifying the reviewer that an assigned request has been resubmitted.
      • Closeout and complete requests.
      • Communicate to requesters and assigners via tracked comments by automated emails.
      • View/add/edit/delete/download documents for certain type of requests.
      • View/add/edit/delete external 3rd-party contacts for certain types of requests.
      • View/add/delete contract numbers for certain types of requests.
      • View/add/edit/delete ProjectWise links for certain types of requests.
      • View/add/remove points/polygons for certain types of requests.
  • Requests Tab
    • View a list of requests submitted by the requester.
    • Create new requests.
    • Filter and search requests based on various criteria (i.e. description, requester, type, status, date, etc.)
    • View request details page by clicking on the sequence number.
      • Submit requests for review.
      • Delete unsubmitted requests.
      • View and edit request general information.
      • Communicate to reviewers and assigners via tracked comments by automated emails.
      • Receive automated emails notifying the requester that a request has been assigned or completed.
      • View/add/edit/delete/download documents for certain type of requests.
      • View/add/edit/delete external 3rd-party contacts for certain types of requests.
      • View/add/delete contract numbers for certain types of requests.
      • View/add/edit/delete ProjectWise links for certain types of requests.
      • View/add/remove points/polygons for certain types of requests.
      • For submitted and assigned requests, unlock requests for editing will resubmit the request and change the request date.
  • All Projects Tab
    • View a list of projects.
    • Filter and search projects based on various criteria (i.e. contract number, project number, firm, description, etc.)
    • Create new projects.
    • View project details page by clicking on the project number.
      • Delete projects.
      • View and edit project general info.
      • Assign project managers to projects.
      • View and edit permit dates.
      • View and edit procurement phase info.
      • View and edit construction phase info.
      • Generate milestone dates by project size and ntp date in construction phase.
      • Override auto-generated milestone dates.
  • Assigned Projects Tab
    • View a list of projects assigned to the project manager.
    • Filter and search projects based on various criteria (i.e. contract number, project number, firm, description, etc.)
    • View project details page by clicking on the project number.
      • View project general info.
      • View and edit permit dates.
      • View and edit procurement phase info.
      • View and edit certain construction phase info.
        • Update finished milestone dates after completion.
        • If a milestone is late, it requires a comment; otherwise comments are optional.
  • Project Report Tab
    • View, filter and print milestone reports.
